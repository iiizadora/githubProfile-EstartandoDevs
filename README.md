- styled-components
- axios
- VS Code 


# ✔ How to use

You need to use Nodejs.

Clone the project

$ git clone [link](git@github.com:iiizadora/githubProfile-EstartandoDevs.git)

Install dependencies

$ npm install or yarn

Start app

$ npm start or yarn start
